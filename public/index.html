Node.js

whiteboard.io

This is a Node.js application for live teacher-student interaction to simulate practical sessions in a college, and could also be used for distance learning. This application was built by me and my team in under 30 hours in a hackathon conducted in VCET college, Mumbai, and got us a bronze medal.

This application allows a teacher to create rooms which students join, and provides the students a code-editor and allows for the compilation and running of their code in this application itself. The teacher can see and edit the code of any student live, and the code can later be posted in a feed which can be viewed by all other students.
